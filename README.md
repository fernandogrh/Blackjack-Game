## Overview

This Blackjack program simulates a simple match between the player and the dealer.
It includes score evaluation, Ace adjustment, dealer rules, and robust user input validation to ensure a smooth gameplay experience.

⸻⸻

## Features

	•	Deal and calculate card values
	•	Automatic Ace conversion (11 → 1 when needed)
	•	Dealer draws until reaching 17
	•	Blackjack detection
	•	Player decisions: hit or stand
	•	Input validation and error handling
	•	Clear and modular function structure
